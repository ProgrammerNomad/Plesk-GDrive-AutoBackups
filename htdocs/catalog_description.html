GDrive AutoBackups for Plesk

Automatically back up your Plesk server data to Google Drive with this easy-to-use extension.

Key Features

  • Automated Backups - Schedule backups to run daily, weekly, or monthly
  • Google Drive Integration - Securely connect your Plesk server to Google Drive
  • Selective Backups - Choose specific directories and databases to back up
  • Retention Policies - Configure how many backups to keep in Google Drive
  • Manual Backup Option - Run backups on demand with a single click
  • Detailed Logs - Track all backup operations and troubleshoot issues

Easy Setup Process

Step 1: Create Google Cloud Project

Set up a project in Google Cloud Console and enable the Drive API

Step 2: Configure API Credentials

Enter your Google API credentials in the extension interface

Step 3: Connect to Google Drive

Authorize the extension to access your Google Drive

Step 4: Configure Backup Settings

Select directories, schedule frequency, and retention policy

Monitor Your Backups

Track all backup activities through the detailed logs interface

View Backups in Google Drive

All your backups are neatly organized in your Google Drive account
